# twitter-behavior-analyzer
Final project for CSE 557A.

App link: https://twitter-behavior-analyzer.onrender.com

Dev Instructions
------
* Install npm/nodejs on your machine
* Clone the repo
* Cd into the repo folder
* Run `npm install`
* Run `node app.js`
* Open a browser to `localhost:3001`
